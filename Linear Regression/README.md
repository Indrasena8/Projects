## Simple Linear implementation with python
This project predicts the salary of a person based on years of experience using Linear Regression

The Salary.csv file contains the two columnls, years of experience and salary

## Dependencies
1. Numpy
2. Matplolib
3. Pandas
4. sklearn

## Installation
Execute the following commands in the command prompt to install the required libraries
* pip install numpy
* pip install matplotlib
* pip install pandas
* pip install sklearn

## Usage
Just run `jupyter notebook` in terminal and you can visit the notebook in your web browser.

Install `jupyter` from [here](http://jupyter.readthedocs.io/en/latest/install.html)
